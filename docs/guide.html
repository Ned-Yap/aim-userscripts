The home base. Install this one first — the others plug into it.

⚙️Control Panel▾

One settings window for all the tools. Turn things on/off, change hotkeys, and paste your GitHub token here.

Open click the ⚙ gear in the map toolbar
Tap for details ▾

How to use it

  1. Click the ⚙ gear in the map toolbar to open it.
  2. Each tool has its own section — flip a switch to turn a feature on or off.
  3. Changed something by accident? Click the ↺ arrow to reset just that setting.

Good for

  • Turning off a tool you're not using right now.
  • Changing a hotkey if it clashes with something else.
  • Pasting your GitHub token (bottom of the panel).

Drawing & measuring on the map

Everyday tools for marking up and measuring the map.

🗺️Map Styler▾

Shows the power-line shielding overlays and checks that everything is covered. The big one most people use daily.

Use Shift+O to toggle
Tap for details ▾

How to use it

  1. Press Shift+O to show or hide the overlays.
  2. Open the Outlines section in the ⚙ panel to turn distribution / transmission lines on or off.
  3. Run the Coverage check to drop numbered pins anywhere that's too far from shielding.

Good for

  • Confirming a site's missions cover all the power lines.
  • Spotting gaps more than ~200 ft from shielding.
  • Recoloring or hiding lines to cut clutter.
📏Measure / Ruler▾

Measure how far it is between points on the map.

Use Shift+R
Tap for details ▾

How to use it

  1. Press Shift+R to start measuring.
  2. Click a start point, then click an end point.
  3. Read the distance; press Shift+R again to exit.

Good for

  • Checking the distance between two assets.
  • Sanity-checking how wide a buffer is.
📍Absolute Altitude▾

Drop pins that show the real altitude at a spot.

Use Shift+A
Tap for details ▾

How to use it

  1. Press Shift+A, then click the map to drop a pin.
  2. The pin shows the altitude there.
  3. Right-click a pin to delete it.

Good for

  • Checking the ground height before setting a mission altitude.
  • Comparing altitude across a few spots.
➕New Entity Macro▾

Create map entities super fast with the number keys instead of clicking through menus.

Use 1–6
Tap for details ▾

How to use it

  1. Press a number 1–6 to instantly start that entity type.
  2. Use the shift shortcuts for actions (e.g. save).
  3. Delete is double-press (Shift+D twice) so you can't do it by accident.

Good for

  • Stamping out lots of entities quickly on a new site.
  • Keeping your hands on the keyboard instead of the menus.
🧹Clear All▾

Wipes all your temporary map drawings in one press.

Use Shift+C
Tap for details ▾

How to use it

  1. Press Shift+C to clear your drawings, measurements, and pins.

Good for

  • Resetting the map after measuring or sketching.
  • Clearing clutter before a screenshot.
🔍

Looking things up

Quickly see details about what's on a site.

🔍Asset Inspector▾

Right-click anything on the map to see its details. Also adds a site-wide summary table and a stats page with charts.

Use right-click an entity, or Shift+Ctrl+Q
Tap for details ▾

How to use it

  1. Right-click any entity for a popup with its name, ID, altitude and notes. Click a number to copy it.
  2. Use Find in Map Entities to jump to it in Percepto's sidebar.
  3. Click SUM (next to ALT/VAL) for the full table; click 📊 for charts and stats.

Good for

  • Grabbing an asset name or altitude in one click.
  • Auditing a whole site at a glance.
  • Exporting the entity list to a spreadsheet (CSV / Sheets).
📋Mission Bank Tools▾

A summary panel and altitude / ground-height tools for the Mission Bank screen.

Use opens in the Mission Bank
Tap for details ▾

How to use it

  1. Open the Mission Bank — the summary panel appears automatically.
  2. Right-click a mission for its details.
  3. Use the batch tools to view/adjust altitude and see ground height per step.

Good for

  • Reviewing all of a site's missions in one place.
  • Checking altitude-above-ground for each step.

Getting around faster

Less clicking, less waiting, fewer wrong pages.

🧭Defaults▾

Sends you to the right page automatically and tidies up map layers on every site.

Use automatic · settings in the ⚙ panel
Tap for details ▾

How to use it

  1. It just works once installed.
  2. In the ⚙ panel, pick the section you want to land on (e.g. Site Setup).
  3. Switching sites keeps you on the same section instead of bouncing to Data Map.

Good for

  • Always landing on your usual page when you open a site.
  • Old "past mission" links auto-jumping to the mission log.
  • Turning off layers you never want, and keeping the Feeder Line on top.
🎮Map Nav▾

Move the map with your keyboard like a game.

Use W A S D · Q/E zoom
Tap for details ▾

How to use it

  1. W A S D to pan, Q/E to zoom out/in.
  2. Hold Alt to move faster (sprint).
  3. Press Space to fit the whole site in view.

Good for

  • Zipping around a big site while reviewing.
⚡

Power-line map data

For the folks who maintain the shielding lines themselves.

⚡Power Line EditorAdvanced▾

Edit the actual power-line map files right on the map and save changes for everyone.

Open the ⚡ button in the map toolbar
Tap for details ▾

How to use it

  1. Click the ⚡ button, then turn on edit mode.
  2. Drag points to move them, draw new lines, or delete lines.
  3. Ctrl+Click a point to branch off it; lines snap to nearby ones.
  4. Save to commit the change for the whole team.

Good for

  • Fixing a power line that's in the wrong place.
  • Adding a new branch or splitting a line.
🚩

Reporting & flagging

Mark problems so the team can track them.

🚩Issues▾

Circle a spot on the map and leave a note about a problem. The whole team sees it, with a status.

Open the 🚩 button in the map toolbar
Tap for details ▾

How to use it

  1. Click the 🚩 button and draw a shape around the problem area.
  2. Add a note and a priority.
  3. Track it through Open → Ready → Resolved (it can be re-opened).

Good for

  • Flagging a coverage gap or data error with its exact location.
  • Keeping a shared to-do list of site fixes.

Setup & behind the scenes

Quietly make things faster, smoother, or easier to manage.

🛡️Performance Shield▾

Blocks background stuff that slows the page down and eats memory. Just makes Percepto snappier.

Use automatic · settings in the ⚙ panel
Tap for details ▾

How to use it

  1. It works automatically once installed.
  2. In the ⚙ panel, the Performance section lets you toggle what's blocked (screen recording, chat, weather).

Good for

  • A laggy or memory-hungry site — keep the background blocks on.
👥Bulk User SitesAdmin▾

On the admin batch page, add users to many sites at once.

Use Shift+B on the admin batch page
Tap for details ▾

How to use it

  1. Open the admin User↔Site batch page and press Shift+B.
  2. Paste or search a list of people and a list of sites.
  3. Click Add all matches; pin favorites and save presets for next time.
  4. Role defaults to Percepto Admin.

Good for

  • Onboarding a batch of new users to lots of sites at once.
