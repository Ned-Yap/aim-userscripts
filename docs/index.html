Get Tampermonkey

It's a free browser add-on that runs these little helper tools. Click your browser:

Chrome / Edge / Brave Firefox Safari

Once it installs, look for the Tampermonkey icon up near your address bar (click the puzzle-piece 🧩 if it's hidden):

Optional: turn on automatic updates
  1. Click the Tampermonkey icon → Dashboard
  2. Settings tab → scroll to Userscript Update
  3. Set Check Interval to Every Day

New versions then show up on their own.

3

Add a GitHub token

A one-time "key" that unlocks the power-line overlays. Don't use those? You can skip this.

A — make a free GitHub account (skip if you have one)
  1. Go to github.com/signup
  2. Use any email (it isn't shown publicly), pick a username like aim-johnd, verify your email
B — get added to the map-files repo
  1. Send Payden your GitHub username
  2. Accept the email invite GitHub sends you
C — create your token
Use a "classic" token (not fine-grained) — fine-grained ones can't read this repo.
  1. Go to github.com/settings/tokens → Tokens (classic)
  2. Generate new token (classic)
  3. Note: AIM KML reader · Expiration: 1 year · check the repo box
  4. Generate, then copy it (starts with ghp_…) — shown once
D — paste it into AIM Controls
  1. Reload Percepto, click the gear (⚙)
  2. Scroll to GitHub Connection (KMLs) → expand → paste → Save & Test
  3. Dot turns green = connected
Overlays not showing within ~10s? Hard reload: Ctrl+Shift+R (⌘+Shift+R on Mac).

More help

Already had older AIM scripts?

Replace them — the old hand-pasted ones don't talk to the Control Panel. In the Tampermonkey Dashboard, delete any script starting with AIM or Percepto (trash icon), then reinstall from Step 2. To be safe, Utilities → Export to zip first.

No gear icon in the map toolbar

Make sure Control Panel is installed and switched on in Tampermonkey, then reload the page.

Hotkeys do nothing

Open the Tampermonkey Dashboard and check each AIM script is enabled.

Things act weird after a bunch of updates

Hard reload: Ctrl+Shift+R (⌘+Shift+R on Mac).

The Install button didn't pop anything up

Copy the link, then in the Tampermonkey Dashboard go to Utilities → Import from URL and paste it.

Report a bug or ask for a feature

Go to the issues page and pick Bug report or Feature request. You'll need access first — same as the map-files step.
